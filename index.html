Introduction to Open Sound Control & SuperCollider IDE Workshop

License: CC BY-SA 4.0

This project is intended to be a project for MUMT 306 - Music & Audio Computing I and also as a Music Tech Workshop hosted by Electrical, Computer, and Software Engineering Student Society at McGill (ECSESS)

Goals

The goal of this project is to introduce

  • The Open Sound Control (OSC) protocol standard
  • SuperCollider (sclang) and the SuperCollider IDE (scide)
  • The client-server architecture of SuperCollider
  • Application of OSC in spatialized and real-time audio synthesis

Project structure

This project contains 2 main parts:

  • A presentation that introduces the OSC protocol and SuperCollider IDE. It containts the step-by-step guide to SuperCollider

  • A simple game that uses SuperCollider as its sound synthesizer for realtime and spatialized audio effects. (Licensed under MIT License)
